<h1>Backup & File Recovery Management</h1>


<h2>Description</h2>
Configured and tested Windows backup and recovery features on a Windows 10 system. Prepared backup storage, configured File History to protect selected directories, performed scheduled backups, and verified successful file restoration and system recovery functionality.
<br />


<h2>Environments Used </h2>

- <b>Windows 10
- DiskPart
- Windows File History
- Volume Shadow Copy Service (VSS)
- System Restore</b>

<h2>Key Tasks Performed</h2>

Created a dedicated backup volume using DiskPart

Configured File History to store backups on a local backup drive

Selected and customized folders included in the backup process

Excluded unnecessary folders from backup scope

Configured automated backup schedule and retention policy

Performed a manual backup operation

Simulated accidental file deletion and restored files from backup

Enabled Volume Shadow Copy Service (VSS)

Used a System Restore Point to recover lost data

<h2>Skills Demonstrated </h2>

- <b>Backup management
- Disaster recovery testing
- Data protection
- System administration
- Windows backup tools</b>
